

Program task2.py will help you when you get lost in internet.


When you are very very hungry and lokking for a restaurant website you can sometimes get very lost.

In case you get lost in internet you will propably see a 404 Error Pages.

But you can change this site to a Are_you_lost.html site (wich is in this folder).

From this site you will get informations what to do in case of get lost.

This site have some information how to creating the map of website.


To use program task2.py write in command line: python3 task2.py url
